# ReadMe
## Ingredients:
- LM35 Sensor
- Sparkfun Serial LCD-09067
- serLCD library from https://github.com/scogswell/ArduinoSerLCD
 
## Instructions:
### LM35 Pins:
- (+) to 3.3v and AREF
- (-) to GND
- (~) Signal to A0
### LCD Pins:
- (+) 5V pin to 5V 
- (-) GND pin to GND
- (RX) RX pin to Digital 2
